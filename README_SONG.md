코드 충돌 (Code conflict):
서로가 같은 코드 파일을 동시에 수정해서(settings.py) 각각의 수정이 충돌이 났습니다.  이런 경우, git은 충돌 부분을 자동으로 감지하고, 사용자에게 충돌 방안을 제시하기 때문에 우리는 git에서 settings.py를 수정했습니다. 

브랜치 (Branch)
